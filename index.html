Default:

jQuery Populaty abstract the name annotation used with dot article.title like VRaptor works or brackets article[title] like Rails does.
The fields select multiple and checkbox can receive an array with objects [{ 'id': 1 }, { 'id': 2 }] or just the raw values [1, 2].
If you wonder about arrays without index like article.categories[].id (VRaptor) or article[category_ids][] (Rails), yes, it works too! (:

The population is based on fields within the form, thus avoiding waste of resource searches across all JSON loaded.
<input type="text" name="title" />

<select name="article.category.id">
  <option></option>
  <option value="1">jQuery Capty</option>
  <option value="2">jQuery Chrony</option>
</select>

<input type="checkbox" name="article[category_ids][]" value="1" />
<input type="checkbox" name="article[category_ids][]" value="2" />
<input type="checkbox" name="article[category_ids][]" value="3" />

<select name="article.categories[].id" multiple="true">
  <option value="flapy">jQuery Flapy</option>
  <option value="flury">jQuery Flury</option>
  <option value="gridy">jQuery Gridy</option>
</select>

<input type="radio" name="article.parent.id" value="1" />
<input type="radio" name="article.parent.id" value="2" />
<input type="radio" name="article.parent.id" value="3" />

<textarea name="article[comment]"></textarea>

var json = {
  'title'  : 'jQuery Raty',
  'article': {
    'categories'  : [{ 'id': 'flapy' }, { 'id': 'gridy' }],
    'category'    : { 'id': 1 },
    'category_ids': [1, 2, 3],
    'comment'     : 'http://wbotelhos.com/labs',
    'parent'      : { 'id': 2 }
  }
};

$('form').populaty({ json: json });

Reset:

something on textarea and then trigger the button Populate.
The value will not be removed because the reset options was disabled.
The value will not be replaced too, because nothing came on JSON for this field. 

$('form').populaty({
  json : { 'title': 'jQuery Raty' },
  reset: false
});

JSON:

The JSON used to populate the fields.
If you set a default JSON for this options, it will be merged with your received JSON. 

$.fn.populaty.defaults.json = { version: '0.1.1' };

$('form').populaty({ json : { 'title': 'jQuery Raty' } });

Include:

Fields that will be handled to be populated.
Check the option exclude to check the fields that will be excluded from this option.
Here we will include just the textarea type, then the text will be ignored. 

$('form').populaty({
  json   : { text: 'jQuery Raty', textarea: 'jQuery Gridy' },
  include: 'textarea'
});

Exclude:

This option will exclude elements included on option include.
He we do the inverse logic used on include options, we exclude the textarea type. 

$('form').populaty({
  json   : { text: 'jQuery Raty', textarea: 'jQuery Gridy' },
  exclude: 'textarea'
});

Typeful:

Typeful is the fields where we type text inside it, different of a select element.
Here we tell the fields that will use the type strategy to be populated. 

<input type="text" name="text_name" />
<input type="score" name="textareaName" />

$('form').populaty({
  json   : { text_name: 'text type as not applied for', textareaName: 'textarea was applied for and matched' },
  typeful: ['textarea']
});

Selectable:

Like typeful, here we tell the selectables fields. 

<select name="select">
  <option></option>
  <option value="1">jQuery Capty</option>
  <option value="2">jQuery Chrony</option>
</select>

<select name="multiple" multiple="true">
  <option value="1">jQuery Flapy</option>
  <option value="2">jQuery Flury</option>
  <option value="3">jQuery Gridy</option>
</select>

$('form').populaty({
  json      : { select: 1, multiple: 1 },
  selectable: ['select-one']
});

Checkable:

Like typeful, we have the checkable inputs too. 

<input type="radio" name="radio" value="1" />
<input type="radio" name="radio" value="2" />

<input type="checkbox" name="checkbox" value="1" />

$('form').populaty({
  json     : { checkbox: 1, radio: 1 },
  checkable: ['radio']
});

Options:

checkable ['checkbox', 'radio']

The checkable fields. This fields will be checked with checked="checked"

exclude ':submit, :reset, :button'

Field to be excluded from includeds.

include ':input'

Fields to be included on population.

json undefined

JSON data to populate the fields. You can to use as String or Object.

reset true

Reset or not the form before populate.

selectable ['select-one', 'select-multiple']

The selectable fields. This fields will be selected with selected="selected"

typeful ['text', 'textarea']

The typeful fields.
